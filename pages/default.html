Default Commands

These are the basic and navigation commands available to all players.

Basic Commands

  • /help [command] – Display a list of available commands or get details on a specific command.
  • /list – Show a list of online players.
  • /msg <player> <message> – Send a private message to a player.
  • /reply <message> – Reply to the last private message received.
  • /me <message> – Send an action message to the chat.
  • /whisper <player> <message> – Send a private message (alias of /msg).

Navigation Commands

  • /spawn – Teleport to the server's spawn point.
  • /home [name] – Teleport to your saved home location.
  • /tpa <player> – Request to teleport to another player.
  • /tpaccept – Accept a teleport request.
  • /tpdeny – Deny a teleport request.
  • /back – Teleport to your previous location.
